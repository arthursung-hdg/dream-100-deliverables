Campaign Setup, Step by Step

  1. Open Ads Manager → Create → Manual setup.
  2. Select objective by funnel stage (below). One campaign per stage so budgets stay isolated.
  3. Name the campaign: Quantis_Launch001_{Stage}_{Pain} (e.g. Quantis_Launch001_Cold_Coordination).
  4. Advantage+ campaign budget, starting $40–$80/day per stage. Raise 20% every 3 days once cost-per-lead stabilizes.
  5. Ad set: conversion location Website (or Instant Form for lead-gen), attribution 7-day click + 1-day view.
  6. Audience: apply the stage profile from the next panel.
  7. Placements: Advantage+ Placements, exclude Audience Network on cold campaigns.
  8. Creative: upload the PNG, paste Primary Text, Headline, Link Description, set CTA, paste the deep link with UTMs.
  9. Pixel: verify Lead and CompleteRegistration fire on the registration thank-you page.
  10. Publish → run each ad set 7+ days or to 50 leads before judging.

Objective by Funnel Stage

  • Cold (Ad 01): Awareness → Reach, or Leads → Landing Page Views.
  • Consideration (Ad 02): Traffic → Landing Page Views.
  • Mid (Ad 03): Leads → CompleteRegistration (webinar sign-up).
  • Bottom (Ad 04): Leads → Schedule (book a call). 14-day learning window.
  • Education (Ad 05): Traffic → Landing Page Views, then retarget engagers.
